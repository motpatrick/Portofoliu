Un cookie este un fișier text care este plasat pe hard disk-ul tău de un server de pagini web. Cookie-urile nu pot fi utilizate pentru a rula programe sau a livra viruși pe computerul tău. Cookie-urile îți sunt atribuite în mod unic și pot fi citite doar de un server web din domeniul care ți-a emis cookie-ul.

Putem utiliza cookie-uri pentru colectarea, stocarea și urmărirea informațiilor în scopuri statistice sau de marketing pentru a opera site-ul nostru web. Ai capacitatea de a accepta sau de a refuza cookie-urile. Majoritatea browserelor web acceptă automat cookie-urile, dar de obicei poți modifica setarea browserului tău pentru a refuza cookie-urile, dacă preferi. Dacă alegi să refuzi cookie-urile, este posibil să nu poți experimenta pe deplin funcțiile interactive ale serviciilor [Numele Site-ului Tău] sau ale site-urilor web pe care le vizitezi.
